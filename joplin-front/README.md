# Joplin-Front : The FrontEnd

## Requirements 

* [VueJS](https://vuejs.org)2 with :
  * Vuex
  * vuex-map-fields
  * axios
  * marked
  * js-cookie
  * vue-infinite-loading

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
