# Joplin Web : The FrontEnd

## Requirements 

* [VueJS](https://vuejs.org) (Vuex, axios)

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
